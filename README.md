# TaskDesk

- TaskDesk is a comprehensive project management
- Task tracking application designed to streamline workflows for teams and individuals.
- It provides a private workspace for organizing work through versatile interfaces,
- Including **Kanban boards**,
- **Hierarchical outlines**, and **Calendars**
- Built for containerized deployment
- TaskDesk ensures a consistent and scalable environment by orchestrating the application,
 database, and cache.

#### Key features 
>- `Project portfolio dashboards` for high-level monitoring 
>- The ability to create `public project boards`
>- External stakeholders to view progress without requiring authentication.
